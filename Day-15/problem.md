# Day 15 - Two Sum II - Input Array Is Sorted

## Problem

Given a 1-indexed sorted array of integers, find two numbers such that they add up to a target.

Return their indices.

## Approach

- Use Two Pointers.
- Place one pointer at the beginning and one at the end.
- Move pointers based on the current sum.

## Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
